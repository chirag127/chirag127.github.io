About Chirag Hub

Chirag Hub is a collection of free, privacy-focused online tools. Every tool is designed to process your files entirely in your browser - nothing is ever uploaded to a server.

Mission

To provide high-quality, free tools that respect user privacy. In a world where most online tools harvest your data, we believe you deserve better.

Tech Stack

All tools are built with vanilla HTML, CSS, and JavaScript. No heavy frameworks, no complex build processes - just clean, performant code that works in any modern browser.

Key libraries used include PDF-lib for PDF manipulation, JSZip for compression, and various Canvas APIs for image processing.
